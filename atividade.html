Paginas HTML

Introdução

Elementos e Tags: No HTML, os elementos são os blocos de construção básicos de uma página web. Cada elemento é representado por tags, como para abrir e para fechar. Por exemplo, define um parágrafo e,fecha o parágrafo.

Estrutura Básica: Toda página HTML começa com para indicar o tipo de documento e como o elemento raiz. Dentro do , temos para metadados (como título, links para estilos) e para o conteúdo visível da página.
